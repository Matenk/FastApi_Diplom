 Task Management

## Описание проекта

Task Management API — это веб-приложение, разработанное с использованием FastAPI, которое позволяет пользователям управлять своими задачами. Пользователи могут создавать, читать, обновлять и удалять задачи. Это приложение обеспечивает высокую производительность и простоту использования благодаря асинхронной обработке запросов.

## Основные компоненты

- **Backend:** FastAPI для обработки запросов и взаимодействия с базой данных.
- **Database:** SQLAlchemy для работы с базой данных.


## Требования

- Python 3.7 или выше
- FastAPI
- SQLAlchemy
- uvicorn (ASGI сервер)
- pip (для установки зависимостей)

## Установка и запуск


Установите необходимые зависимости:
bash
pip install -r requirements.txt

Запустите приложение:
bash
uvicorn app.main:app --reload

Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/.

Задачи
GET /tasks: Получение списка всех задач
POST /tasks: Создание новой задачи
PUT /tasks/{task_id}: Обновление существующей задачи
DELETE /tasks/{task_id}: Удаление задачи
